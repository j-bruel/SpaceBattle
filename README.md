# SpaceBattle

*Project for learnings purpose during the Epitech student years.*

_______

The goal of this project is to create a new online game based on standard naval battle across network.

Game built with two application :

Client : C++14 application with 3D GUI (GUI built with Urho3D).

Server : C++14 application without GUI (Linux) and custom network protocol.

Few technologies used : boost, Urho3D, googletest, plog, msgpack.
